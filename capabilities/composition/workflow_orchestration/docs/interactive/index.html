Quick Start Guide

Get up and running with APG Workflow Orchestration in minutes

Installation

Install APG Workflow Orchestration using pip or Docker

pip install apg-workflow-orchestration
apg-workflow init
apg-workflow start
docker run -p 5000:5000 \
  -e DATABASE_URL=postgresql://... \
  -e REDIS_URL=redis://... \
  datacraft/workflow-orchestration:latest
kubectl apply -f https://raw.githubusercontent.com/datacraft/apg-workflow/main/k8s/deployment.yaml
